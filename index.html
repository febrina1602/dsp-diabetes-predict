Deteksi Dini Risiko
Diabetes Anda

Sistem cerdas bertenaga Machine Learning untuk menganalisis metrik kesehatan dan memprediksi status glikemik secara akurat.

💡
Tentang Sistem Ini: DiaCare menggunakan model prediktif berbasis data klinis untuk mengklasifikasikan status glikemik ke dalam tiga kategori: Normal, Pre-Diabetes, dan Diabetes. Hasil bersifat indikatif dan tidak menggantikan diagnosis medis profesional.
🤖

Model Machine Learning

Prediksi berbasis model yang dilatih dengan dataset klinis riil mencakup 5.922 pasien untuk akurasi tinggi.

🧪

14 Biomarker Klinis

Menganalisis kombinasi tekanan darah, enzim hati, indeks resistensi insulin, dan marker metabolik lainnya.

📊

Dashboard Analitik

Visualisasi data interaktif yang menampilkan distribusi, korelasi, dan tren epidemiologi diabetes Indonesia.

⚡

Respons Instan

Hasil komputasi diagnosa dikirimkan secara real-time melalui koneksi ke server AI lokal.

🎯

3 Kelas Prediksi

Output berupa PGclass3: Normal (0), Pre-Diabetes (1), atau Diabetes (2) dengan warna indikator yang jelas.

🔒

Data Privat & Lokal

Input tidak disimpan. Semua pemrosesan dilakukan secara lokal melalui server Flask di perangkat Anda.
